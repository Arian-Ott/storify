Welcome to Storify: Your Personal File Server

Storify is a powerful, lightweight, and self-hosted static file server designed to give you complete control over your digital content. Built with FastAPI, it offers a seamless way to upload, manage, and share your files securely and efficiently.

How Storify Works

At its core, Storify focuses on efficiency and security. When you upload a file, Storify intelligently calculates its hash (a unique digital fingerprint). This allows it to check if an identical file already exists on the server. If a match is found, instead of creating a redundant copy, Storify creates a symbolic link (symlink). This smart deduplication technique ensures that only one physical copy of any given file is ever stored, saving valuable disk space and streamlining management.

For secure access, Storify leverages JSON Web Tokens (JWT) for user authentication and authorization. This robust system ensures that only authorized users can access protected resources, such as personal files, profile pictures, or sensitive user-specific data, providing a high level of data integrity and privacy.

Key Features

  • Fast & Simple Hosting: Effortless management and serving of static files.
  • Secure Access: User authentication and authorization via robust JWTs.
  • Smart Deduplication: Optimized storage using file hashing and symbolic links.
  • Protected Resources: Controlled access to sensitive user data.
