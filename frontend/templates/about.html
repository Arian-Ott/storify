About Storify

Storify is a lightweight file storage server built with FastAPI, designed to let users upload and share files with ease. It offers efficient, self-hosted static file hosting while ensuring secure access and minimal duplication.

When a file is uploaded, Storify calculates its hash and checks whether it already exists on the server. If a match is found, instead of storing the file again, a symbolic link (symlink) is createdâ€”ensuring only one copy is retained, saving valuable storage space.

The platform supports user authentication and authorisation via JSON Web Tokens (JWT), enabling access control to protected resources like profile pictures or user-specific data.

Key Features

  • Fast and simple hosting of static files
  • User authentication for secure file management
  • Smart deduplication using file hashing
  • Protected resources requiring valid JWT access
